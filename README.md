# Got Key'd Realty - Commission Tracker

A desktop app for tracking agent commissions, generating invoices, and producing tax documents.

## What It Does

**Commission Invoices** — Enter a sale, pick the agent, and the app calculates the commission split automatically based on each agent's contract terms. It generates two PDF invoices: one for the office's records and one to give to the agent.

**Cap Tracking** — See at a glance how close each agent is to hitting their annual cap. Once an agent is capped, they keep 100% of their commission (or 50/50 on company leads). The cap year is based on each agent's contract anniversary date, not the calendar year.

**Tax Documents (1099-NEC)** — At the end of the year, pull up any agent's total compensation and generate their 1099 form with one click. The app tracks earnings from every sale automatically — no need to add anything up manually.

**Agent Management** — Add new agents, update contract terms, and manage split types (percentage, tiered, or flat transaction fee).

## How to Install

### Windows
1. Go to the [Releases](https://github.com/ainthony1/gkr/releases) page
2. Download the latest `.zip` file
3. Unzip it and double-click **GotKeyd Commission Tracker.exe**

That's it — no other software needed.

### Mac
1. Go to the [Releases](https://github.com/ainthony1/gkr/releases) page
2. Download the Mac `.zip` file
3. Unzip it and drag the app to your Applications folder

## Where Things Get Saved

- **Database** — Stored in your computer's app data folder (you don't need to touch this)
- **Invoices & 1099s** — Saved to `Documents/GotKeyd Realty/` so they're easy to find
